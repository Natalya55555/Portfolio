# project vision

## Проект. Определение возраста покупателей по фото
                               
Построение модели, которая по фотографии определит приблизительный возраст человека. Фотофиксация в прикассовой зоне поможет определять возраст клиентов.

## Стеки

- **python**
- **pandas**
- **numpy**
- **matplotlib**
- **keras**
